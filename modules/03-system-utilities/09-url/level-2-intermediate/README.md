# Level 2: URL Intermediate

Advanced URL manipulation techniques and patterns.

## Learning Objectives

By completing this level, you will:

- ✅ Build URLs programmatically with complex requirements
- ✅ Master URLSearchParams for advanced query operations
- ✅ Work with relative URLs and path resolution
- ✅ Implement URL encoding and decoding correctly
- ✅ Handle different URL protocols and schemes
- ✅ Compare modern URL API with legacy methods
- ✅ Build reusable URL utilities

---

## Prerequisites

- Completed Level 1: Basics
- Understanding of URL structure
- Familiarity with URLSearchParams

---

## What You'll Learn

### Core Topics

1. **Programmatic URL Construction**
   - Building URLs from templates
   - Dynamic endpoint generation
   - URL builders and factories

2. **Advanced Query Parameters**
   - Complex parameter structures
   - Nested objects in queries
   - Array and multi-value handling
   - Query string serialization

3. **Relative URL Resolution**
   - Understanding relative paths
   - Resolving URLs against bases
   - Path normalization

4. **URL Encoding/Decoding**
   - When and how to encode
   - Different encoding methods
   - Special character handling

5. **Protocol Handling**
   - Working with file:// URLs
   - Custom protocols
   - Protocol-relative URLs
