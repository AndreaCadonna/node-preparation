# Level 1: Examples

This directory contains 8 practical examples demonstrating the basics of the OS module.

## Examples Overview

1. **01-basic-system-info.js** - Basic system information retrieval
2. **02-memory-information.js** - Memory monitoring and formatting
3. **03-cpu-information.js** - CPU details and metrics
4. **04-user-information.js** - User info and directory paths
5. **05-system-uptime.js** - System uptime tracking
6. **06-platform-constants.js** - EOL and endianness
7. **07-combined-system-info.js** - Comprehensive system report
8. **08-practical-use-cases.js** - Real-world applications

## How to Run

Execute any example with Node.js:

```bash
node 01-basic-system-info.js
```

## Learning Path

Work through the examples in order for the best learning experience.
