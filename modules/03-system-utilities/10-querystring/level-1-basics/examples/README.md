# Level 1 Examples

This directory contains practical code examples demonstrating query string fundamentals.

## Examples

1. **[01-basic-parsing.js](01-basic-parsing.js)** - Parse query strings into objects
2. **[02-basic-stringifying.js](02-basic-stringifying.js)** - Convert objects to query strings
3. **[03-url-encoding.js](03-url-encoding.js)** - Handle URL encoding and decoding
4. **[04-type-handling.js](04-type-handling.js)** - Work with different data types
5. **[05-url-integration.js](05-url-integration.js)** - Combine with URL module
6. **[06-practical-examples.js](06-practical-examples.js)** - Real-world use cases

## Running Examples

Each example is a standalone Node.js script. Run them with:

```bash
node 01-basic-parsing.js
node 02-basic-stringifying.js
# etc...
```

## What You'll Learn

- Query string syntax and format
- Parsing and stringifying methods
- URL encoding for special characters
- Type conversions (strings to numbers, booleans)
- Array handling in query strings
- Integration with URL module
- Real-world patterns and use cases
