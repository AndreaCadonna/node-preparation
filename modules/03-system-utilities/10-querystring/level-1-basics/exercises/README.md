# Level 1 Exercises

Practice query string fundamentals with these hands-on exercises.

## Exercises

1. **exercise-1.js** - Basic parsing and stringifying
2. **exercise-2.js** - URL encoding and special characters
3. **exercise-3.js** - Type conversion and validation
4. **exercise-4.js** - Building URLs with parameters
5. **exercise-5.js** - Search functionality (comprehensive)

## How to Complete

1. Open each exercise file
2. Read the task descriptions
3. Implement the TODO sections
4. Run the file to test: `node exercise-1.js`
5. Check your output against expected results
6. Compare with solutions after attempting

## Learning Objectives

- Parse query strings to objects
- Convert objects to query strings
- Handle URL encoding properly
- Convert string values to correct types
- Build complete URLs dynamically
- Handle arrays and special cases
- Create reusable URL builder functions
