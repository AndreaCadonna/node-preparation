# Level 1 Exercises

Practice HTTP basics with these hands-on exercises.

## Instructions

1. Read each exercise carefully
2. Implement the solution
3. Test your code thoroughly
4. Compare with solutions (after attempting)

## Exercises

1. **exercise-1.js** - Basic Server and Routing
2. **exercise-2.js** - Query Parameters
3. **exercise-3.js** - POST Request Handling
4. **exercise-4.js** - HTTP Client
5. **exercise-5.js** - Simple Static File Server

## Running Exercises

```bash
node exercise-1.js
```

Good luck!
