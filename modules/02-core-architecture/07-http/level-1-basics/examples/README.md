# Level 1 Examples

This directory contains 8 practical examples demonstrating HTTP basics in Node.js.

## Running the Examples

```bash
# Run any example
node 01-basic-server.js

# Visit in browser
# http://localhost:3000

# Or test with curl
curl http://localhost:3000
```

## Examples Overview

1. **01-basic-server.js** - Creating your first HTTP server
2. **02-request-info.js** - Working with request properties
3. **03-routing.js** - Implementing basic routing
4. **04-status-codes.js** - Using different status codes
5. **05-request-body.js** - Reading and parsing request bodies
6. **06-http-client.js** - Making HTTP client requests
7. **07-static-files.js** - Serving static files
8. **08-json-api.js** - Building a simple JSON API

Each example is fully commented and runnable. Study the code, run it, and experiment!
