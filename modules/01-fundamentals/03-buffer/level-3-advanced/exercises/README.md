# Level 3 Exercises

Practice exercises for advanced buffer operations.

## Exercises

### Exercise 1: High-Performance Log Parser
Build a streaming parser for binary log files with millions of entries.

**Topics**: Streaming, buffer pooling, performance optimization

### Exercise 2: Memory-Efficient Image Processor
Process large image files without loading entirely into memory.

**Topics**: Chunked processing, zero-copy operations, memory management

### Exercise 3: Binary Protocol Fuzzer
Create a tool to test binary protocol implementations.

**Topics**: Binary data generation, edge cases, security testing

### Exercise 4: Buffer Pool Implementation
Design and implement a production-grade buffer pool.

**Topics**: Resource management, performance, memory profiling

### Exercise 5: Secure File Processor
Build a file processor with comprehensive security checks.

**Topics**: Security validation, error handling, safe operations

## Instructions

1. Read the exercise specification
2. Design your solution
3. Implement with tests
4. Profile performance
5. Compare with reference solutions

## Running Exercises

```bash
cd level-3-advanced/exercises
node exercise-1.js
```
