# Level 2 Examples

Code examples demonstrating intermediate buffer operations.

## Examples List

1. **01-numeric-types.js** - Reading/writing integers and floats
2. **02-endianness.js** - Little-endian vs big-endian examples
3. **03-typed-arrays.js** - Working with TypedArrays and ArrayBuffer
4. **04-bmp-parser.js** - Parsing BMP image headers
5. **05-wav-parser.js** - Parsing WAV audio headers
6. **06-network-packet.js** - Building network packets
7. **07-binary-protocol.js** - Complete binary protocol implementation
8. **08-file-format-writer.js** - Creating binary file formats

## How to Run

```bash
cd level-2-intermediate/examples
node 01-numeric-types.js
```

Each example includes:
- Detailed comments
- Console output
- Practical use cases
- Error handling
