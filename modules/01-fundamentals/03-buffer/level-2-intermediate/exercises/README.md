# Level 2 Exercises

Practice exercises for intermediate buffer operations.

## Exercises

### Exercise 1: WAV File Parser
Parse WAV audio file headers to extract metadata.

**Topics**: Binary file formats, multi-byte integers, endianness

### Exercise 2: TCP Packet Builder
Implement a TCP-like packet format with header and checksum.

**Topics**: Binary protocols, network byte order, checksums

### Exercise 3: TypedArray Converter
Build utilities to convert between different TypedArray types.

**Topics**: TypedArrays, data type conversion, buffer interop

### Exercise 4: Binary Serializer
Create a serializer for JavaScript objects to binary format.

**Topics**: Binary encoding, type encoding, variable-length data

### Exercise 5: Network Protocol Implementation
Implement a complete binary protocol with framing and error detection.

**Topics**: Protocol design, encoding/decoding, error handling

## Instructions

1. Read the exercise file
2. Implement the required functions
3. Test with provided test cases
4. Verify against expected output
5. Compare with solutions

## Running Exercises

```bash
cd level-2-intermediate/exercises
node exercise-1.js
```
