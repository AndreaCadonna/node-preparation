# Level 1 Exercises

Practice exercises for Buffer basics.

## Exercises

### Exercise 1: Buffer Creation and Conversion
Create buffers from various sources and convert between formats.

**Topics**: Buffer creation, string encoding/decoding

### Exercise 2: Working with Binary Data
Read a binary file and extract specific byte values.

**Topics**: File I/O, byte access, hexadecimal

### Exercise 3: Buffer Comparison
Implement functions to compare buffers and find differences.

**Topics**: Buffer comparison, equality checking

### Exercise 4: Concatenating Buffers
Build a buffer assembler that efficiently joins chunks.

**Topics**: Buffer concatenation, performance

### Exercise 5: Encoding Converter
Create a tool to convert text between different encodings.

**Topics**: Character encodings, encoding conversion

## Instructions

1. Read the exercise file
2. Implement the required functions
3. Run the tests to verify your solution
4. Compare with solutions after attempting

## Running Exercises

```bash
cd level-1-basics/exercises
node exercise-1.js
```
