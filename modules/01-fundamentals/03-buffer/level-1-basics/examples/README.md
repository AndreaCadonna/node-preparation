# Level 1 Examples

Code examples demonstrating Buffer basics.

## Examples List

1. **01-buffer-creation.js** - Creating buffers with different methods
2. **02-string-conversion.js** - Converting between strings and buffers
3. **03-encodings.js** - Working with different character encodings
4. **04-byte-operations.js** - Reading and writing individual bytes
5. **05-buffer-comparison.js** - Comparing buffers
6. **06-buffer-concat.js** - Concatenating multiple buffers
7. **07-binary-files.js** - Reading and writing binary files
8. **08-practical-use-cases.js** - Real-world buffer applications

## How to Run

```bash
cd level-1-basics/examples
node 01-buffer-creation.js
```

Each example is self-contained and includes detailed comments explaining the concepts.
